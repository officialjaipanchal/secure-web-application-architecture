How does AWS work?

  • Search your required AWS Service

    AWS provides thousands of different services.Search for service which you require and select it. For example, AWS S3, AWS CloudWatch

  • Setup required configuration

    Each AWS service requires different configuration. Set necessary configuration before using it. For example, If using AWS S3 then setup bucket name etc.

  • Pay and Use

    Depending on AWS tier pay required costs for usage of your services and start using AWS. Most of AWS services works on pay per usage model.
